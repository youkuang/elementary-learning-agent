## 核心架构

本系统采用多 Agent 架构,每个学科一个专门的 Agent,加上一个协调器 Agent。

### Agent 职责

- **语文 Agent**: 处理生词、古诗、课文等语文学习任务
- **数学 Agent**: 处理口算、应用题、公式等数学学习任务
- **英语 Agent**: 处理单词、句型、语法等英语学习任务
- **协调器 Agent**: 跨学科协调、生成综合报告、发送提醒

### 数据流

```
家长输入 → Agent 解析 → 任务分解 → 知识点追踪 → 
错误分析 → 自适应教学 → 复习调度 → 进度报告
```

### 核心算法

1. **间隔重复算法**: 基于错误次数动态调整复习频率
2. **自适应教学**: 根据错误模式选择不同的教学策略
3. **知识点分解**: 将学习任务拆分为可独立追踪的知识点
